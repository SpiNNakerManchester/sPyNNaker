Detailed Description

interface for neurons

The API contains:

  • neuron_initialise(): translate the data stored in the NEURON_PARAMS data region in SDRAM and converts it into C-based objects for use.
  • neuron_set_input_buffers(input_buffers_value): setter for the internal input buffers
  • neuron_do_timestep_update(): executes all the updates to neural parameters when a given timer period has occurred.

translate the data stored in the NEURON_PARAMS data region in SDRAM and convert it into c based objects for use.

Parameters
[in]addressthe absolute address in SDRAM for the start of the NEURON_PARAMS data region in SDRAM
[in]recording_addressthe recording parameters in SDRAM (contains which regions are active and how big they are)
[out]n_neurons_valueThe number of neurons this model is to simulate
[out]n_synapse_types_valueThe number of synapse types in the model
[out]incoming_spike_buffer_sizeThe number of spikes to support in the incoming spike circular buffer
[out]n_rec_regions_usedThe number of regions used by neuron recording
Returns
True if the translation was successful, otherwise False
