Detailed Description

interface for different weight implementations for the weight half of a STDP rule.

the API interface contains:

  • weight_initialise(): Initialises the weight aspect of a STDP rule.
  • weight_get_initial(): Converts the stored weight into an update-able form.
  • weight_get_final(): Converts the updated weight into store-able form.
