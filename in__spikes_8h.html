static bool in_spikes_initialize_spike_buffer ( uint32_t  size)
inlinestatic

This function initializes the input spike buffer.

It configures:

buffer: the buffer to hold the spikes (initialized with size spaces) input: index for next spike inserted into buffer output: index for next spike extracted from buffer overflows: a counter for the number of times the buffer overflows underflows: a counter for the number of times the buffer underflows

If underflows is ever non-zero, then there is a problem with this code.

Parameters
[in]sizeThe number of spikes we expect to handle in the buffer
Returns
True if the buffer was successfully initialised
