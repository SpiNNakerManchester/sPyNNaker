property structs¶
The structures of the component. If there are multiple structs,

the order is how they will appear in memory; where there are structs that repeat per neuron the repeats will appear adjacent e.g. for non-repeating struct g, followed by repeating structs s1 and s2 with 3 neurons the layout will be: [g, s1, s1, s1, s2, s2, s2].

Return type:

list(Struct)
