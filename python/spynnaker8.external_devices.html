spynnaker8.external_devices package¶

Module contents¶

This contains functions and classes for handling external devices such as the PushBot (http://spinnakermanchester.github.io/docs/push_bot/).

Note

When using external devices, it is normally important to configure your SpiNNaker system to run in real-time mode, which usually reduces numerical accuracy to gain performance.
